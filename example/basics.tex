  \item ちゃお
  \item りぼん
  \item なかよし
\end{itemize}

これは番号を振る箇条書きです。

\begin{enumerate}
  \item 富士
  \item 鷹
  \item なすび
\end{enumerate}

これは語句説明の箇条書きです。

\begin{description}
  \item[遷移クリープ] クリープ速度が次第に減少
  \item[定常クリープ] クリープ速度がほとんど時間的に変化しない
  \item[加速クリープ] 破断するまで連続的にクリープ速度が増加
\end{description}
